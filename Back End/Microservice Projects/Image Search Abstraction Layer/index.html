FreeCodeCamp API Basejump: Image Search Abstraction Layer

      User Story: I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
      User Story: I can paginate through the responses by adding a ?offset=2 parameter to the URL.
      User Story: I can get a list of the most recently submitted search strings.

Search for an image to receive data. There are different routes you can use.

  • '/' : returns the home page you're viewing now.
  • '/api/imagesearchall' : returns all searches stored in the database.
  • '/api/imagesearch/:searchterm*' : returns json data containing alt text, a thumbnail url, and the website hosting this content.
  • '/api/latestimagesearch' : returns the 10 most recent searches.
  • Every other route returns a 404 status.
