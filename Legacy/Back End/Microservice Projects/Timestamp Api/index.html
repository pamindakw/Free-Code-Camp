Objective: Build a fullstack javascript app that returns json formatted timestamp data based on what is entered in the URL.


      User Story: I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016).
      User Story: If it does, it returns both the Unix timestamp and the natural language form of that date.
      User Story: If it does not contain a date or Unix timestamp, it returns null for those properties.

Example unix input: https://timestamp-api-vinarius.herokuapp.com/1502323200

Will return the following JSON data: {"unix":1502323200,"natural":"August 10, 2017"}


Conversely, entering a natural language date: https://timestamp-api-vinarius.herokuapp.com/August 10, 2017

I get: {"unix":1502323200,"natural":"August 10, 2017"}


Erroneous input: https://timestamp-api-vinarius.herokuapp.com/bacon

I get: {"unix":null,"natural":null}
